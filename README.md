# Uzay Atış Oyunu

Godot 4.6 ile yapılmış basit bir uzay atış oyunu — masaüstü ve web tarayıcısında çalışır.

## 🎮 Oyna

**Web sürümü:** https://RmznBDY.github.io/uzay-atis-oyunu/

## Kontroller

| Cihaz | Hareket | Ateş |
|---|---|---|
| Klavye | ⬅️ ➡️ | Boşluk |
| Mouse / Touch | Tıkla & sürükle | Otomatik (basılı tut) |

## Özellikler

- Yıldızlı arka plan (paralaks)
- Patlama efekti
- 3 can sistemi
- Vurulan düşmanlar yerine hayvan emoji'si bırakır 🦊🐱🐰

## Yerel Geliştirme

[Godot 4.6](https://godotengine.org/download/windows/) ile `project.godot` dosyasını aç.
